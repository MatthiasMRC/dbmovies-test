## Documentation

**Framework :** Flutter

**Details packages :**
- [dio](https://pub.dev/packages/dio) handle http request (REST API)
- [intl](https://pub.dev/packages/intl) string manipulation (format date)
- [provider](https://pub.dev/packages/provider) State Management
